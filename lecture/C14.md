# 14.*抽象数据类型



## 14.1 动态数组

1. 可自增长
2. 可访问当前长度
3. 可以快速访问并且修改其中的任意一个元素

## 14.2 单双链表

```c
// 单链表
struct node {
    int val;
    struct node *next;
};
// 双链表
struct list {
    struct node *head;
    struct node *tail;
};
```

## 14.3 二叉树

```c
struct node {
    int val;
    struct node * left;
    struct node * right;
};
```
