# 注释

1. 代码中帮助人理解的部分, 并无实际作用

2. 单行注释

   ```c
   # include <stdio.h>
   
   int main(void) {
   	// 这是单行注释
   	printf("Hello World!\n");
   	
   	return 0;
   }
   ```

3. 多行注释

   ```c
   # include <stdio.h>
   
   int main(void) {
   	/*
   		这是多行注释
   	*/
   	printf("Hello World!\n");
   	
   	return 0;
   }
   ```
