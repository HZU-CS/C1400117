# 条件运算符

| 运算符 | 意义     |
| ------ | -------- |
| ==     | 相等     |
| !=     | 不相等   |
| >      | 大于     |
| >=     | 大于等于 |
| <      | 小于     |
| <=     | 小于等于 |

```c
#include <stdio.h>

int main(void) {
    
    printf("%d\n", 5 == 3);
    printf("%d\n", 5 != 3);
    printf("%d\n", 5 > 3);
    printf("%d\n", 5 >= 3);
    printf("%d\n", 5 < 3);
    printf("%d\n", 5 <= 3);
    
    return 0;
}
```
